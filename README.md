# node-todos
Learning Node, Express, and Testing

*Requirements:* You need to have NodeJS installed on your machine.

### Running

*To get started:*
```
  npm install
```

*run server one time:*
```
  npm start
```

*hot reloading:*
```
  npm run dev
```

### Testing

*One time test*
```
  npm test
```

*Test watch*
```
  npm run test-watch
```
